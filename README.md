# 🌿 EcoLens AI | Sustainability Data & ROI Predictor

**EcoLens AI** is an enterprise-grade ESG (Environmental, Social, and Governance) dashboard designed to help companies track carbon footprints and simulate the financial return of green energy investments.

## 🚀 Key Features
* **Real-time Dashboard:** Visualizes $CO_2$ emissions across facilities with industry benchmarking.
* **AI-Powered Insights:** Uses LLMs to generate executive sustainability summaries and action plans.
* **ROI Simulator:** A strategic tool for executives to calculate the payback period of solar, wind, or EV transitions.
* **Secure Data Entry:** Modular data input system with built-in unit conversion and validation.

## 🛠️ Tech Stack
* **Frontend/App Framework:** Streamlit
* **AI Engine:** Groq LPU / LangChain
* **Version Control:** Git & GitHub
* **Environment:** Python 3.10+

## 📂 Project Structure
* `app.py`: The "Lead Architect" landing page and system status.
* `pages/`: Multi-page application structure for Dashboard, ROI, and AI Reports.
* `utils/`: Backend logic for carbon calculations and AI prompt engineering.
